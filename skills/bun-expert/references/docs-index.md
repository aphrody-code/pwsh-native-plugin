## Guides (task-oriented recipes — 100+)

Available under `/guides/`:
- `binary/` — ArrayBuffer / Buffer / Blob / typed-array conversions
- `deployment/` — AWS Lambda, DigitalOcean, Google Cloud Run, Railway, Render, Vercel
- `ecosystem/` — Astro, Discord.js, Docker, Drizzle, Elysia, Express, Hono, Mongoose, Next.js, Nuxt, PM2, Prisma, Qwik, React, Remix, Sentry, SolidStart, SvelteKit, systemd, TanStack Start, Upstash, Vite
- `html-rewriter/` — link / meta extraction
- `http/` — cluster, fetch, unix sockets, file uploads, hot reload, proxy, server, SSE, streaming, TLS
- `install/` — dep types, CI/CD, custom registries, migration, Azure Artifacts, JFrog, workspaces
- `process/` — argv, CTRL+C, IPC, spawn, stdin, signals, uptime
- `read-file/` / `write-file/` — every format conversion
- `runtime/` — constants, GitHub Actions, macOS codesign, env vars, imports, timezone, tsconfig paths, VS Code / web debugger
- `streams/` — Node readable + Web stream conversions
- `test/` — bail, concurrent, coverage, happy-dom, Jest migration, mock clock, spies, snapshots, timeouts, watch
- `util/` — base64, deflate, gzip, deep-equals, uuid, path / file URL, sleep, upgrade, version, which
- `websocket/` — compression, context, pub/sub, simple server

Full list at <https://bun.com/docs/guides/index.md>.
